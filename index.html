Privacy Policy

Quiz Helper

1. Introduction

Quiz Helper is a browser extension designed to help users quickly select answers using keyboard shortcuts.

2. Data Collection

We do NOT collect, store, or share any personal data.

3. How the Extension Works

The extension only operates locally in your browser. It may:

  • Detect answer options on a webpage
  • Allow selection using keyboard shortcuts
  • Automatically navigate to the next question

4. Data Sharing

We do not share any data with third parties.

5. Permissions

The extension may request access to webpage content in order to function properly. All processing is done locally on your device.
