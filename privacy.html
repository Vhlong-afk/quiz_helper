# Privacy Policy for Quiz Helper ## 1. Introduction Quiz Helper is a browser extension designed to help users select answers quickly using keyboard shortcuts. ## 2. Data Collection Quiz Helper does NOT collect, store, or share any personal data. ## 3. How the Extension Works The extension only interacts with the content of web pages locally in your browser to: * Detect answer options * Allow keyboard-based selection * Automatically move to the next question ## 4. Data Sharing We do not share any user data with third parties. ## 5. Permissions The extension may request access to web pages in order to function properly, but all processing is done locally on your device. ## 6. Changes This policy may be updated in the future. Continued use of the extension means you accept the updated policy. ## 7. Contact If you have any questions, please contact us at: [your-email@example.com](mailto:your-email@example.com)
